# Student Record Management System (Python)

A clean and lightweight console-based application built in Python for managing student records. This project focuses on building strong programming fundamentals, file handling skills, and clean code practices.

## Features

- Admin Login System
- Add, Update, Delete Student Records
- Automatic Grade Calculation
- Persistent Data Storage using JSON
- Input Validation and Error Handling
- Sorting by Marks
- Display Top Performing Students
- Clean and Modular Code Structure

## Technologies Used

- Python 3
- JSON (File-based Storage)
- Git & GitHub

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/mhuzaifhamid/student-record-management-system.git
```

2. Navigate to the project folder:

```bash
cd student-record-management-system
```

3. Run the application:

```bash
python main.py
```
